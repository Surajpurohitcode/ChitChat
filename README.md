# ChitChat

ChitChat real-time chat application for android using firebase.

## Tech Stack
#### Technologies
Android, Java

#### Firebase
* Authentication
* Realtime Database
* Storage



## Features

- Realtime chats updates
- phone auth system
- update profile images and bio
- lock your chats
